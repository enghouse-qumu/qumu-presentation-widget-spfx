# Qumu PresentationWidget for SharePoint

This is a SharePoint web part that integrates the Qumu Presentation Widget.

![Qumu Presentation Widget](assets/readme.png)

### Installation

- Download [qumu-presentation-widget-spfx.sppkg](sharepoint/solution/qumu-presentation-widget-spfx.sppkg) file.
- Upload it to your **Apps for SharePoint** catalog.
- Add the **Qumu Presentation Widget** web part to your SharePoint page.
- Configure it with your Qumu Host, presentation GUID and Playback mode.
